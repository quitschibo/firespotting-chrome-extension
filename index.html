Firespotting Chrome Extension

This is an extension for Google Chrome that displays the latest ideas from Firespotting. It also features desktop notifications of the current #1 idea. It follows the philosophy, that you don't have to open a tab or a additional window to get updates from firespotting.

This is a fork from Adam Albrecht's Hacker News Chrome Extension - Thank you very much.

Features

  • Notifies about #1 idea
  • Marks "fired" ideas
  • Opens tabs in background, if you like

How to install

Just take the extension from google chrome store
